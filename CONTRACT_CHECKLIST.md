# CONTRACT CHECKLIST

## ✅ BEFORE SIGNING ANY CONTRACT

### Compensation
- [ ] Rate specified (€/hr or fixed price)
- [ ] Payment terms (Net 15/30)
- [ ] Late payment penalty (1.5% monthly)
- [ ] Expense reimbursement process

### Scope & Deliverables
- [ ] Detailed SOW with specific deliverables
- [ ] Timeline/milestones defined
- [ ] Change request process
- [ ] Clear exit criteria

### Intellectual Property
- [ ] Pre-existing IP excluded (your code/methods)
- [ ] Work product ownership defined (client owns deliverables)
- [ ] Portfolio rights included (you can showcase work)

### Termination
- [ ] 2-4 weeks notice (mutual)
- [ ] No penalty for termination
- [ ] Payment for work completed

### Liability
- [ ] Cap at contract value or €50k
- [ ] Client indemnifies you for their misuse

### Other
- [ ] Independent contractor status (not employee)
- [ ] Governing law: Spain or neutral
- [ ] No non-compete clauses
- [ ] Right to list work in portfolio

---

## 🚨 RED FLAGS

- ❌ No portfolio rights
- ❌ Unlimited liability
- ❌ Net 60+ payment terms
- ❌ Non-compete restriction
- ❌ No termination notice

---

**Never start work without signed contract.**
